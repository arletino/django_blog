Функционал сайта:


  1. Регистрация и авторизация.
    Автор - это зарегистрированный пользователь с возможностью добавления и публикации рецептов на сайте

  2. Просмотр всех опубликованных рецептов по различным категориям всех авторов. Возможно для всех пользователей

  3. Просмотр автором всех своих рецептов (опубликованных/неопубликованных)

  4. Возможность публикации и снятия с публикации автором всех своих рецептов

  5. Добавление рецепта. Возможно только для автора данного рецепта

  6. Изменение рецепта. Возможно только для автора данного рецепта

  7. Удаление рецепта. Возможно только для авторов данного рецепта

  8. Создание комментариев к рецепту. Возможно для всех пользователей

  9. Просмотр комментариев к рецепту. Возможно для всех пользователей

  10. Расчет количества просмотров и средней оценки рецепта

  11. Организация обратной связи




    12. Сайт разработан на Django python v.5.1.1, HTML5/CSS3 Титков Аркадий 2024г.
